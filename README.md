# Liora for Typora

Liora 是一个专为 Typora 设计的简洁优雅主题，旨在提供舒适的阅读和写作体验。它通过自定义字体、颜色方案和排版细节，让您的 Markdown 文档焕然一新。

## 特点

- **优雅的排版**：采用 "Futura-Medium" 等字体，优化了段落、标题、列表等元素的间距和样式，提升可读性。
- **清晰的色彩**：定义了 `--text-color`、`--heading-color`、`--accent` 等变量，确保文本、标题和强调色的和谐统一。
- **代码高亮**：为行内代码和代码块提供了清晰的背景色和字体样式，便于代码阅读。
- **引用和链接优化**：美化了引用块和超链接的显示效果，使其更具辨识度。
- **图片居中显示**：图片默认居中显示，并带有圆角效果，提升视觉美观度。

## 安装

1.  **下载主题**：

将 `liora.css` 文件下载到您的本地计算机。

2.  **打开 Typora 主题文件夹**：

- 在 Typora 中，点击 `偏好设置` (或 `Preferences`)。
- 在 `外观` (或 `Appearance`) 选项卡中，点击 `打开主题文件夹` (或 `Open Theme Folder`)。

3.  **复制主题文件**：

将下载的 `liora.css` 文件复制到打开的主题文件夹中。

4.  **应用主题**：

- 重启 Typora。
- 点击 `主题` (或 `Themes`) 菜单，选择 `Liora` 即可应用。
